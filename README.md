# 線上測驗系統 - OnlineTest
> 練習測驗檢定學科的工具
> 提供出作業、錯誤歷史、分組競賽等功能

## 一、應用技術
* HTML
* JS
* CSS
* ASP .NET
* SQL Server

## 二、特色功能
* 題庫管理
* 測驗指派
* 錯誤題目統整

## 三、系統說明
* 受到高中老師的委託，幫助學弟妹們建立了一個練習測驗檢定學科的工具
    * 教師管理：題庫、作業、考試、錯誤題目報表、團體競賽
    * 學生練習：計時測驗、不計時練習、錯誤題目整理
